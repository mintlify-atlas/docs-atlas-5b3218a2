## Payroll settings

### GET /api/hr/payroll-settings

Return the current payroll configuration (deduction rules, allowance settings, etc.).

```bash
curl http://localhost:3003/api/hr/payroll-settings \
  -H "Authorization: Bearer <access_token>"
```

### PATCH /api/hr/payroll-settings

Update payroll configuration. Send only the fields you want to change.

## Payroll

### GET /api/hr/payroll-sheet

Return the payroll sheet (all staff salary data) for a given month.

<ParamField query="month" type="string" required>
  Month in `YYYY-MM` format (e.g. `2025-03`).
</ParamField>

<ParamField query="type" type="string" required>
  Staff type: `teacher`, `employee`, or `all`.
</ParamField>

```bash
curl "http://localhost:3003/api/hr/payroll-sheet?month=2025-03&type=all" \
  -H "Authorization: Bearer <access_token>"
```

### GET /api/hr/payroll-history

Return payroll history for a specific staff member.

<ParamField query="staffId" type="number" required>
  Staff member ID.
</ParamField>

<ParamField query="type" type="string" required>
  Staff type: `teacher`, `employee`, or `all`.
</ParamField>

### POST /api/hr/payroll

Create or update (upsert) a payroll entry for a staff member.

<Note>
  The request body shape mirrors the payroll record: staffId, month, basicPay, allowances, deductions, netPay, status, etc. The exact fields depend on the payroll settings configuration.
</Note>

```bash
curl -X POST http://localhost:3003/api/hr/payroll \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{"staffId": 5, "month": "2025-03", "basicPay": 50000, "netPay": 47000, "status": "PAID"}'
```

## Staff attendance

### GET /api/hr/staff-attendance

Return staff attendance records for a specific date.

<ParamField query="date" type="string" required>
  Date in ISO format (e.g. `2025-03-15`).
</ParamField>

```bash
curl "http://localhost:3003/api/hr/staff-attendance?date=2025-03-15" \
  -H "Authorization: Bearer <access_token>"
```

### POST /api/hr/staff-attendance

Mark attendance for a staff member. Requires authentication; the marking admin's ID is recorded automatically.

<ParamField body="staffId" type="number">
  Unified staff member ID.
</ParamField>

<ParamField body="date" type="string" required>
  Attendance date (ISO string).
</ParamField>

<ParamField body="status" type="string" required>
  Attendance status (e.g. `PRESENT`, `ABSENT`, `LEAVE`).
</ParamField>

<ParamField body="notes" type="string">
  Optional notes.
</ParamField>

```bash
curl -X POST http://localhost:3003/api/hr/staff-attendance \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{"staffId": 5, "date": "2025-03-15", "status": "PRESENT"}'
```

### GET /api/hr/attendance-summary

Get a monthly attendance summary for a staff member.

<ParamField query="month" type="string" required>
  Month in `YYYY-MM` format.
</ParamField>

<ParamField query="staffId" type="number" required>
  Staff member ID.
</ParamField>

<ParamField query="type" type="string" required>
  Staff type (currently supports `TEACHER`).
</ParamField>

## Holidays

### POST /api/hr/holidays

Create a holiday entry.

<ParamField body="date" type="string" required>
  Holiday date (ISO date string).
</ParamField>

<ParamField body="title" type="string" required>
  Holiday title (e.g. `Pakistan Day`).
</ParamField>

<ParamField body="type" type="string" required>
  Holiday type (e.g. `National`, `Religious`, `Gazetted`).
</ParamField>

<ParamField body="repeatYearly" type="boolean">
  If `true`, this holiday recurs on the same date every year. Defaults to `false`.
</ParamField>

<ParamField body="description" type="string">
  Optional description.
</ParamField>

```bash
curl -X POST http://localhost:3003/api/hr/holidays \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{"date": "2025-03-23", "title": "Pakistan Day", "type": "National"}'
```

### GET /api/hr/holidays

List holidays, optionally filtered by year and month.

<ParamField query="year" type="number">
  Filter by year.
</ParamField>

<ParamField query="month" type="number">
  Filter by month (1–12).
</ParamField>

### DELETE /api/hr/holidays

<ParamField query="id" type="number" required>
  Holiday ID.
</ParamField>
