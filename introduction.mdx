## What is Concordia PSH?

Concordia PSH is a full-featured institution management platform designed for intermediate colleges, undergraduate programs, and short-course institutes. It unifies every administrative function into a single, permission-controlled system — from student enrollment and fee collection to staff payroll and hostel allocation.

The platform is built on a **NestJS** backend with **Prisma ORM** and **MySQL**, paired with a **React** admin panel. The entire stack is containerized with **Docker Compose** for straightforward deployment.

## Role-based access control

Every admin account can be scoped to specific modules. A SUPER_ADMIN has unrestricted access, while regular ADMINs are granted access only to the modules they need — right down to individual features within each module.

```json
{
  "role": "ADMIN",
  "permissions": {
    "modules": ["Students", "Fee Management", "Attendance"]
  }
}
```
